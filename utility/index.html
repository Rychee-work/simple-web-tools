Utility Tools

Everyday generators and converters that save time.

About this category

Utility tools are small helpers for common daily tasks — the kind of things you’d normally search for “quickly convert”, “generate”, or “calculate”. These tools are designed to be fast, easy to understand, and safe to use. Many run entirely in your browser so your inputs aren’t uploaded to a server.
Typical use cases include generating UUIDs for development, converting Unix timestamps for logs and APIs, creating strong passwords, and generating QR codes. Pick a tool below to open it, read the short guide at the top of the page, and run the tool right away.

How to use these tools

Open a tool, read the short explanation at the top, then run it in your browser. For more categories, use the sidebar.
